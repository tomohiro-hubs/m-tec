# 株式会社エムテック（m-tec）窓フィルム施工 LP プロジェクト

## プロジェクト概要
- **プロジェクト名**: 株式会社エムテック 窓フィルム施工サービス LP
- **目的**: 関西圏（大阪・兵庫・京都・奈良・和歌山・滋賀）の個人住宅・法人向けに、低価格・高品質な窓フィルム施工を訴求し、公式LINEからの問い合わせ・見積もり誘導を行う。
- **デザインコンセプト**: 清潔感、信頼感、プロフェッショナル。現代的な日本のビジネスイラストスタイル（細い紺色の線画）とクリーンなレイアウト。

## 現在の実装状況
### デザイン・ビジュアル
- **統一されたイラストスタイル**: CTAセクション、サイズガイド、ホスピタリティ（家具移動）、解決バナーなど、すべてのビジュアルをオリジナルのビジネスイラストに統一。
- **プロフェッショナルな配色**: ネイビー、ライトブルー、ホワイトを基調とした、清潔感と信頼感のあるカラーパレット。
- **モバイルファースト**: スマートフォンでの操作性と視認性を最優先したレスポンシブデザイン。
- **マイクロインタラクション**: 各ボタンやステップカードにプレミアム感を演出するアニメーション（ホバー効果、シャイン効果等）を実装。

### 主要セクション
1. **ヒーローセクション**: 印象的なコピーとバッジによる主要メリットの提示。
2. **お悩み解決セクション**: 暑さ、視線、結露、防災の4つの視点からメリットを解説。
3. **1m²サイズガイド**: 一般的な窓の大きさを視覚的に理解しやすく解説。
4. **価格表（コミコミ定額）**: 透明性の高い価格提示。
5. **エムテックの強み**: 自社施工による低価格と3年保証の訴求。
6. **ホスピタリティ**: 家具移動無料などの細やかな配慮。
7. **施工事例**: 実績のグリッド表示。
8. **LINE無料見積もりCTA**: ステップ形式で分かりやすい問い合わせ導線。

### 外部連携
- **公式LINE**: 全ボタン/リンクを実運用アカウント（`https://line.me/R/ti/p/@287hfqga`）に接続済み。
- **Instagram**: 公式アカウントへのリンクを実装済み。

## 開発・技術情報
### 使用技術
- **HTML5**: セマンティックなマークアップ。
- **CSS3 (Vanilla)**: 変数（CSS Variables）を使用した柔軟なデザインシステム。
- **JavaScript**: スクロールアニメーション、スティッキーヘッダーの制御。
- **Icons**: Font Awesome 6.x / 4.x

### ファイル構成

- `css/style.css`: スタイル定義（デザインシステム含む）
- `js/main.js`: インタラクション制御
- `img/`: 施工写真および独自イラスト素材
