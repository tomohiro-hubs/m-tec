# 窓フィルム施工 コミコミ定額ランディングページ

## プロジェクト概要
- **プロジェクト名**: 窓フィルム施工 コミコミ定額ランディングページ
- **目的**: 戸建て・マンションの個人オーナーに向けて、完全定額・低価格・高品質・安心感を訴求し、LINE無料見積もりへ誘導すること
- **主な特徴**:
  - モバイルファーストの1ページ構成LP
  - 信頼性を高める実績・資格・保証訴求
  - 不安除去に特化した導線設計
  - LINE問い合わせへのスムーズな誘導

## 現在実装済みの機能
1. ユーザー提供の施工写真を反映し、可読性を改善したファーストビュー（ヒーローセクション）
2. 悩み訴求セクション
3. コミコミ価格表セクション
4. 安さの根拠と安心材料セクション
5. 家具移動無料の不安解消セクション
6. LINE無料見積もりCTAセクション
7. スティッキーヘッダー
8. スムーズスクロール
9. スクロール時の簡易表示アニメーション
10. レスポンシブデザイン
